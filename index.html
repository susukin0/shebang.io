What is Shebang?

Shebang is a Artix-Runit an Artix GNU/Linux based distribution offering a great blend of end user and power user experience. Using the nimble Openbox window manager, it is highly customisable and provided a modern, full-featured GNU/Linux system without sacrificing performance.
